This handbook is set in Spectral, a transitional serif designed by Production Type for screens. Body sections render at 18 pixels with a 1.85 line-height, generous enough that you can read for an hour without your eye getting tired. The italics throughout the chapter titles and apparatus are Spectral's own.

Sans-serif passages (most of the chrome you don't notice) are set in Inter, a workhorse sans by Rasmus Andersson. Code, where it appears, is JetBrains Mono.

The site itself runs on **Tangly**, an open-source, self-hosted documentation framework, with the *pith* theme: an editorial counterpart to Tangly's default. Tangly themes are small CSS-token packages on top of a shared component library, meaning a handbook like this and an API reference can share the same content engine without compromising either.

If you'd like to assemble something similar, the [project lives on GitHub](https://github.com/tanglydocs/tangly).
