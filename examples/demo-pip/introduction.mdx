---
title: Sprig
description: Tag your shell history. Find it later.
---

Sprig is a tiny CLI that tags shell commands so you can find them again without a regex.

```bash
$ deploy.sh prod  # tag: deploy, prod
$ sprig find prod
2024-08-12  deploy.sh prod
2024-09-02  rollback prod --version v1.2
```

It's about 200 lines of Go. There is no daemon. There is no config file. There is one binary and one history file.

If that's the size of tool you're looking for — the rest of these pages are short.
