# Components

All components are auto-injected into MDX — no imports required. Prop signatures match Mintlify exactly.

## Full list

| Component | Use for |
|---|---|
| `Card`, `CardGroup` | Click-through cards |
| `Note`, `Tip`, `Warning`, `Info`, `Check`, `Danger` | Callouts |
| `Tabs`, `Tab` | Tabbed content |
| `Steps`, `Step` | Numbered procedures |
| `Accordion`, `AccordionGroup` | Collapsible Q&A |
| `CodeGroup` | Multi-language code |
| `Frame` | Image with caption |
| `Tooltip` | Inline hover tip |
| `Icon` | Lucide icon by name |
| `Badge` | Inline label |
| `Expandable` | Inline disclosure (used inside `ParamField`) |
| `ParamField`, `ResponseField` | API doc rows |
| `RequestExample`, `ResponseExample` | API request/response panels |
| `Update` | Changelog entry (label/description/tags) |
| `Snippet` | Reusable MDX (Phase 2) |
| `Columns` | Free-form grid |
