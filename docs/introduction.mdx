# Tangly

Tangly is a self-hosted documentation framework. It reads `docs.json` + MDX, ships sharper authoring primitives on top, and produces a static site that drops onto Vercel or Cloudflare with zero config.

<Note>
  Tangly is Mintlify-compatible: any existing `docs.json` + MDX project renders without source
  edits. Hot-reload everything — including `docs.json` — in under a second.
</Note>

## Why Tangly

Mintlify nailed authoring DX. But it's hosted, closed-source, and you can't extend it without leaving. Tangly keeps the authoring shape — the schema, the components, the conventions — and runs it on Astro, your machine, your repo, your CDN.

## What's included

- **CLI**: `dev`, `build`, `preview`, `init`, `check`.
- **Default theme (Tang)**: Mintlify-Mint inspired, light + dark, Tailwind v4.
- **MDX components**: Card, CardGroup, Note, Tip, Warning, Tabs, Steps, Accordion, ParamField, ResponseField, OpenApi viewer, and more.
- **Schema**: Full Zod schema for `docs.json` with editor-driven validation via `$schema`.
- **OpenAPI**: Pages with `openapi: METHOD path` frontmatter auto-render the endpoint.

## Status

Tangly is in early preview. Phase 1 (rendering Mintlify projects) is functional. Phase 2 (component shadowing, drafts, content collections, custom templates) is next.
