# Custom page templates

<Warning>Custom templates are **Phase 2**. This is the planned shape.</Warning>

Set `template:` in frontmatter to point at a file in `./templates/`:

```md
---
title: Welcome
template: landing
---
```

Tangly resolves to `./templates/landing.astro` and renders the page through it.

## Template props

```ts
interface TemplateProps {
  frontmatter: Frontmatter;
  manifest: Manifest;
  page: PageEntry;
  children: AstroComponent; // the rendered MDX content
}
```

## Section-level inheritance

Set `template` on a group node in `docs.json` to apply it to all descendant pages:

```json
{
  "group": "API",
  "template": "api-overview",
  "pages": [...]
}
```

Page-level frontmatter overrides group-level.

## Falling back

If the named template doesn't exist in `./templates/`, Tangly falls back to the default theme layout and logs a warning.
