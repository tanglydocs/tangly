# Note

`<Note>` is the most common callout. Use it for context that's important but not urgent — caveats, environment requirements, references to related docs.

## When to use

- "This page assumes you've completed the [Quickstart](/quickstart)."
- "Tangly v0.2 is the minimum for this feature."
- Side-references that would clutter the prose.

If you're writing "be careful — this can break things," reach for [`<Warning>`](/reference/components/warning) instead. If it's actively dangerous, use [`<Danger>`](/reference/components/danger).

## Props

None. The component takes children only.

## Example

```mdx
<Note>This page assumes Bun ≥ 1.1.</Note>
```

<Note>This page assumes Bun ≥ 1.1.</Note>

## Customizing

All seven callouts share an internal `_Callout.astro` shell. To restyle every callout in one go, [shadow](/guides/customization/custom-components) `theme/_Callout.astro`. To restyle just `<Note>`, shadow `theme/Note.astro`.
