# Danger

`<Danger>` is the loudest callout Tangly ships. Reserve it for moments when the reader could lose work or compromise security.

## When to use

- "`rm -rf dist/` runs immediately. Don't paste this if the directory contains anything you didn't generate."
- "Never commit your API key to the repo, even if `.env` is gitignored — it leaks via `git log` once tracked."
- Anything where the consequence is hours of recovery, not a re-run.

If the reader can recover by re-running the command, use [`<Warning>`](/reference/components/warning) instead.

## Props

None.

## Example

```mdx
<Danger>Setting `redirects` with the wrong source/destination order will redirect every page to itself, looping the browser.</Danger>
```
