# OpenApiRedoc

`<OpenApiRedoc>` embeds the [ReDoc](https://redocly.com/redoc) standalone bundle, lazy-loaded from CDN. Use it when you'd rather present the operation in ReDoc's three-panel layout than Tangly's compact default.

Set `api.viewer: "redoc"` in `docs.json` to make every auto-generated OpenAPI page use ReDoc by default.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `spec` | `string` | — | Path or URL to the OpenAPI spec. Required. |
| `method` | `string` | — | HTTP method. Required. |
| `path` | `string` | — | Operation path. Required. |

## Example

```mdx
<OpenApiRedoc spec="/openapi.json" method="POST" path="/v1/orders" />
```
