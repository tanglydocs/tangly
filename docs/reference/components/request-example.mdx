# RequestExample

`<RequestExample>` marks one or more code blocks as the request example for the current API operation. The Tangly OpenAPI viewer pulls these out of the page flow and renders them in a side panel next to the description.

## Props

None. Wrap one or more code blocks (typically a [`<CodeGroup>`](/reference/components/code-group)) inside.

## Example

````mdx
<RequestExample>
  ```bash cURL
  curl -X POST https://api.example.com/v1/users \
    -H "Authorization: Bearer $TOKEN" \
    -d '{"name":"Ada"}'
  ```
  ```ts TypeScript
  await fetch("https://api.example.com/v1/users", {
    method: "POST",
    headers: { Authorization: `Bearer ${token}` },
    body: JSON.stringify({ name: "Ada" }),
  })
  ```
</RequestExample>

<ResponseExample>
  ```json 201
  { "id": "u_abc", "name": "Ada" }
  ```
</ResponseExample>
````
