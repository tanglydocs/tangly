# Components

Tangly auto-injects ~35 MDX components into every page. No imports, no configuration. Drop the tag in, render.

Every component listed here lives in [`@tanglydocs/theme-ui`](https://github.com/tanglydocs/tangly/tree/main/packages/theme-ui/src/components). Override any of them by [shadowing the file](/guides/customization/custom-components) at `theme/<Name>.astro` in your project.

## Authoring custom components

Need something we don't ship? Drop a `.astro` file under `theme/` in your project — every MDX page can use it as if it were built-in. See [Custom components](/guides/customization/custom-components).
