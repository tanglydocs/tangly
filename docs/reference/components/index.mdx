## Markdown extras

Looking for math or diagrams? Those are MDX syntax, not components, but they're wired by default. See [LaTeX](/reference/markdown#latex) and [Mermaid](/reference/markdown#mermaid).

## Source of truth

Every component lives in [`@tanglydocs/theme-ui`](https://github.com/tanglydocs/tangly/tree/main/packages/theme-ui/src/components). Override any of them by [shadowing](/guides/customization/custom-components) the file at `theme/<Name>.astro` in your project.

## Authoring custom components

Need something we don't ship? Drop a `.astro` file under `components/` (or `theme/`) in your project. Every MDX page can use it as if it were built-in. See [Custom components](/guides/customization/custom-components).
