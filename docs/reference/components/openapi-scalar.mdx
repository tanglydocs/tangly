# OpenApiScalar

`<OpenApiScalar>` embeds [Scalar](https://scalar.com)'s standalone bundle, lazy-loaded from CDN. Single-operation rendering is approximated by pre-filtering the spec down to the chosen op (Scalar doesn't expose a per-operation prop).

Set `api.viewer: "scalar"` in `docs.json` to make every auto-generated OpenAPI page use Scalar.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `spec` | `string` | — | Path or URL to the spec. Required. |
| `method` | `string` | — | HTTP method. Required. |
| `path` | `string` | — | Operation path. Required. |

## Example

```mdx
<OpenApiScalar spec="/openapi.json" method="DELETE" path="/v1/orders/{id}" />
```
