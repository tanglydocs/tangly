# Card

A `<Card>` is a self-contained tile. It carries a title, an optional icon or hero image, body text, and (usually) a link. Cards stack into a grid via [`<CardGroup>`](/reference/components/card-group).

## When to use

- Landing page entry points ‚Äî point readers at the four or five paths that matter.
- "What's next" footers at the bottom of long guides.
- Visual indexes inside reference sections.

If a card has no link, you're probably reaching for the wrong component ‚Äî try a [Note](/reference/components/note) or plain prose instead.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `title` | `string` | ‚Äî | Bold heading line. |
| `icon` | `string` | ‚Äî | Icon name (Lucide or Mintlify Font Awesome alias). Rendered in a tinted square. |
| `href` | `string` | ‚Äî | If set, the card becomes an anchor. Hover state activates. |
| `horizontal` | `boolean` | `false` | Image left, content right. Without it, image stacks above. |
| `img` | `string` | ‚Äî | Hero image URL. Absolute (`/images/foo.png`) or external. |
| `image` | `string` | ‚Äî | Mintlify-compatible alias of `img`. Either works. |
| `imageAspect` | `string` | `"16/9"` | CSS aspect ratio for the image area. |
| `color` | `string` | ‚Äî | Override icon tint and hover accent. Any CSS color. |

## Examples

### Default

```mdx
<Card title="Quickstart" icon="zap" href="/quickstart">
  Scaffold a project and see it render in under a minute.
</Card>
```

```mdx

  Swap the palette, change the font, ship it.
</Card>
```
