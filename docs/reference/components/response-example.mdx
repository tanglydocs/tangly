# ResponseExample

`<ResponseExample>` marks one or more code blocks as the response example for the current API operation. Renders next to the request example in the side panel.

## Props

None. Wrap code blocks inside.

## Example

````mdx
<ResponseExample>
  ```json 200
  { "id": "u_abc", "name": "Ada" }
  ```
  ```json 422
  { "error": "validation", "fields": ["name"] }
  ```
</ResponseExample>
````

The leading status code on the language line (`200`, `422`) becomes the tab label.
