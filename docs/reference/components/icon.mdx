# Icon

`<Icon>` renders a single inline icon. The default library is [Lucide](https://lucide.dev) — pass any Lucide icon name. Common Mintlify / Font Awesome names are aliased automatically (`circle-check` → `CircleCheck`, `info-circle` → `Info`, `gear` → `Settings`, etc).

Cards, callouts, and steps all accept an `icon` prop that goes through the same resolver.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `icon` | `string` | — | Icon name. Required. Case-insensitive. |
| `size` | `number \| string` | `16` | Pixel size or any CSS length. |
| `color` | `string` | (currentColor) | Override the SVG `stroke`/`fill`. Any CSS color. |
| `class` | `string` | `""` | Extra classes for layout. |

## Examples


## Picking a library

Set `icons.library` in `docs.json`:

```json
{ "icons": { "library": "lucide" } }
```

Valid values: `"lucide"` (default), `"fontawesome"`. See [Schema → Icons](/reference/schema/icons).
