# LightboxImage

`<LightboxImage>` wraps an `<img>` in a clickable button. Click to expand to fullscreen; Esc or outside-click to close. Mobile pinch-zoom works through CSS `touch-action`.

You usually don't write this component by hand — Tangly's MDX renderer wraps every inline `![](...)` in `<LightboxImage>` automatically. Reach for the explicit form when you want to override its props.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `src` | `string` | — | Image URL. Required. |
| `alt` | `string` | `""` | Alt text. |
| `width` | `number \| string` | — | Pass-through to `<img width>`. |
| `height` | `number \| string` | — | Pass-through to `<img height>`. |
| `lightbox` | `boolean` | `true` | Set `false` to disable the modal — renders a plain `<img>`. |

## Opting out

Two ways to render an image without a lightbox:

```mdx
<LightboxImage src="/images/icon.png" alt="" lightbox={false} />
```

Or, inside a [`<Frame>`](/reference/components/frame), set `lightbox={false}` on the Frame and the inner image inherits.
