# AccordionGroup

`<AccordionGroup>` wraps multiple [`<Accordion>`](/reference/components/accordion) elements so they share borders and feel like one component instead of a stack of disconnected boxes.

## Props

None.

## Example

```mdx
<AccordionGroup>
  <Accordion title="Does Tangly support versions?" icon="git-branch">
    Yes — `navigation.versions` lets you author multiple parallel doc trees.
  </Accordion>
  <Accordion title="Does Tangly support i18n?" icon="languages">
    Yes — `navigation.languages` works the same way.
  </Accordion>
  <Accordion title="Does Tangly support PDFs?" icon="file">
    No. Static HTML only.
  </Accordion>
</AccordionGroup>
```
