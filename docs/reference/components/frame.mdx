# Frame

`<Frame>` wraps a single visual element — an image, video, embed, or screenshot — and gives it a thin border with rounded corners. Add a `caption` for figure-style use.

## When to use

- Screenshots that need a "Figure 1" affordance.
- Photos with attribution.
- Standalone diagrams in long-form prose.

For embeds (YouTube, Loom, CodeSandbox), use [`<Embed>`](/reference/components/embed) — it handles iframe sizing for you.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `caption` | `string` | — | Subtitle rendered under the framed content. |

## Examples

### Captioned image

```mdx
<Frame caption="The default tang theme on a docs landing page.">
  ![Tangly docs](/images/screenshot.png)
</Frame>
```

### Plain frame around a diagram

```mdx
<Frame>
  ![Architecture overview](/images/architecture.png)
</Frame>
```
