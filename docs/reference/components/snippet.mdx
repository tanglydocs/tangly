# Snippet

`<Snippet>` references a shared MDX fragment. Author the fragment in `snippets/<name>.mdx`, drop the tag wherever you want it rendered.

<Note>Snippet resolution from disk is on the v0.3 roadmap. Today, `<Snippet>` renders any inline children passed via `<slot />`. To share content across pages now, use [`<Embed>`](/reference/components/embed) — which references a labeled block on another page.</Note>

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `file` | `string` | — | Snippet path relative to `snippets/`. Required. |

## Future shape

```mdx
<Snippet file="install.mdx" />
```

Resolves to `snippets/install.mdx` at build time.
