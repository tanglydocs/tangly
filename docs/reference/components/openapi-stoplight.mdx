# OpenApiStoplight

`<OpenApiStoplight>` embeds [Stoplight Elements](https://stoplight.io/open-source/elements), lazy-loaded from CDN.

Set `api.viewer: "stoplight"` in `docs.json` to use Stoplight for every auto-generated OpenAPI page.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `spec` | `string` | — | Path or URL to the spec. Required. |
| `method` | `string` | — | HTTP method. Required. |
| `path` | `string` | — | Operation path. Required. |

## Example

```mdx
<OpenApiStoplight spec="/openapi.json" method="GET" path="/v1/users" />
```
