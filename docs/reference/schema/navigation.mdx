# `navigation`

Required. Defines the sidebar, tabs, and any global navigation. Recursive — groups can contain groups, tabs can contain anchors, anchors can contain pages.

```json
{
  "navigation": {
    "tabs": [...],
    "global": { "anchors": [...] }
  }
}
```

## Top-level shape

| Field | Type | Description |
|---|---|---|
| `pages` | `NavNode[]` | Flat page list. Use when you don't need tabs/groups. |
| `groups` | `NavGroup[]` | Sidebar groups at the root. |
| `tabs` | `NavTab[]` | Top tabs. Each tab has its own sidebar. |
| `anchors` | `NavAnchor[]` | Pinned anchors. |
| `dropdowns` | `NavDropdown[]` | Pinned dropdowns. |
| `versions` | `NavVersion[]` | Version selector — multiple parallel doc trees. |
| `languages` | `NavLanguage[]` | Language selector. |
| `global` | `GlobalNav` | Persistent across tabs/versions: `{ anchors, dropdowns, tabs }`. |

A `NavNode` is one of: a page slug (`string`), a `NavGroup`, a `NavTab`, a `NavAnchor`, a `NavDropdown`, a `NavVersion`, a `NavLanguage`.

## Page slugs

Page slugs are paths (no leading slash, no `.mdx` extension). They map to files under your docs root:

```json
"pages": ["introduction", "guides/quickstart", "reference/cli/dev"]
```

…matches `introduction.mdx`, `guides/quickstart.mdx`, `reference/cli/dev.mdx`.

## `NavGroup`

```json
{
  "group": "Get Started",
  "pages": ["introduction", "installation", "quickstart"]
}
```

| Field | Type | Description |
|---|---|---|
| `group` | `string` | Required. Sidebar heading. |
| `pages` | `NavNode[]` | Required. Children. |
| `icon` | `string \| { name, library? }` | Heading icon. |
| `root` | `string` | Page slug used when the group's heading itself is clicked. |
| `tag` | `string` | Small chip rendered next to the heading (`"Beta"`, `"Pro"`). |
| `expanded` | `boolean` | Pre-expand on first paint. |
| `template` | `string` | Override the default group-page template. |

Groups can nest: a `NavGroup` may contain other `NavGroup` entries inside `pages`.

## `NavTab`

```json
{
  "tab": "Reference",
  "groups": [
    { "group": "CLI", "pages": [...] },
    { "group": "Schema", "pages": [...] }
  ]
}
```

| Field | Type | Description |
|---|---|---|
| `tab` | `string` | Required. Tab label. |
| `pages` | `NavNode[]` | Optional. Top-level pages in this tab's sidebar. |
| `groups` | `NavGroup[]` | Optional. Grouped pages in this tab's sidebar. |
| `anchors` | `NavAnchor[]` | Optional. Anchors pinned in this tab. |
| `icon` | `string \| { name, library? }` | Tab icon. |
| `href` | `string` | External URL. Tab becomes an outbound link. |
| `openapi` | `string` | Spec URL. Generates OpenAPI pages under this tab. |
| `template` | `string` | Override the default tab template. |

## `NavAnchor`

A pinned navigation item — usually external (GitHub, Twitter, status page).

```json
{ "anchor": "GitHub", "href": "https://github.com/...", "icon": "github" }
```

| Field | Type | Description |
|---|---|---|
| `anchor` | `string` | Required. Display label. |
| `href` | `string` | Target URL. |
| `icon` | `string \| { name, library? }` | Icon. |
| `pages` | `NavNode[]` | If set, anchor becomes a navigable section. |
| `groups` | `NavGroup[]` | Same as above for grouped pages. |

## `NavDropdown`

Same shape as `NavAnchor` but renders as a dropdown menu instead of a link.

```json
{
  "dropdown": "Products",
  "icon": "boxes",
  "groups": [
    { "group": "Tangly", "pages": ["product/tangly"] },
    { "group": "Schema", "pages": ["product/schema"] }
  ]
}
```

## `NavVersion`

Adds a version selector. Each version carries its own page tree.

```json
{
  "versions": [
    { "version": "v2", "default": true, "tabs": [...] },
    { "version": "v1", "tabs": [...] }
  ]
}
```

| Field | Type | Description |
|---|---|---|
| `version` | `string` | Required. Display label and URL prefix. |
| `default` | `boolean` | Marks the default version. |
| `pages`, `groups`, `tabs` | — | Same as the top-level `navigation`. |

## `NavLanguage`

Same shape as `NavVersion` but for i18n.

```json
{
  "languages": [
    { "language": "en", "tabs": [...] },
    { "language": "ja", "tabs": [...] }
  ]
}
```

## `global`

Items rendered persistently across every tab/version/language.

```json
{
  "global": {
    "anchors": [
      { "anchor": "Status", "href": "https://status.example.com", "icon": "activity" }
    ],
    "dropdowns": [],
    "tabs": []
  }
}
```
