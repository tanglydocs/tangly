# Code blocks

Tangly highlights code with [Shiki](https://shiki.style). Every fenced block carries a copy button by default.

## Fenced blocks

Three or more backticks open and close. Language hint goes after the opening fence.

````md
```ts
const greeting = "hello"
```
````

```ts
const greeting = "hello"
```

## File titles

Add `title="..."` after the language hint. Renders a chip above the code block with a file icon derived from the extension.

````md
```ts title="src/main.ts"
import { Tangly } from "tangly"
```
````

```ts title="src/main.ts"
import { Tangly } from "tangly"
```

## Line highlighting

`{1,3-5}` after the language hint highlights specific lines.

````md
```ts {2,4-5}
const a = 1
const b = 2
const c = 3
const d = 4
const e = 5
```
````

```ts {2,4-5}
const a = 1
const b = 2
const c = 3
const d = 4
const e = 5
```

## Line numbers

`showLineNumbers` toggles a number gutter:

````md
```ts showLineNumbers
const a = 1
const b = 2
```
````

Combine with line highlighting:

````md
```ts showLineNumbers {2}
const a = 1
const b = 2
```
````

## Diff syntax

Use `diff` as the language to render +/- lines:

````md
```diff
- const a = 1
+ const a = 2
  const b = 3
```
````

```diff
- const a = 1
+ const a = 2
  const b = 3
```

For a TypeScript diff that keeps syntax highlighting, prefix with `// [!code ++]` and `// [!code --]` Shiki transforms (Tangly bundles the standard set):

````md
```ts
const a = 1 // [!code --]
const a = 2 // [!code ++]
const b = 3
```
````

## Theme

Pick the Shiki theme via [`code.theme`](/reference/schema/fonts-styling) in `docs.json`:

```json
{
  "code": {
    "theme": { "light": "github-light", "dark": "github-dark" }
  }
}
```

## Multi-language tabs

For "same code in multiple languages," wrap blocks in [`<CodeGroup>`](/reference/components/code-group):

````mdx
<CodeGroup>
  ```ts title="fetch.ts"
  await fetch("https://api.example.com")
  ```
  ```py title="fetch.py"
  requests.get("https://api.example.com")
  ```
</CodeGroup>
````

## Disabling the copy button

Globally:

```json
{ "code": { "copyButton": false } }
```

There's no per-block override today.

## Inline code

Single backticks: `` `tangly check` ``.

Use inline code for: function names, file paths, env vars, CLI commands, version numbers, URLs that look better unstyled.
