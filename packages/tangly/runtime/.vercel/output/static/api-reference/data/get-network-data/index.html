API ReferenceGet Network Data

Get Network Data

Get time series data for a network.

GET /v4/data/network/{network_code}

Get Network Data

Get time series data for a network.

Parameters

  • network_code (path) required · string
  • metrics (query) required · array

    The metrics to get data for

  • interval (query)

    The time interval to aggregate data by

  • date_start (query)

    Start time for the query

  • date_end (query)

    End time for the query

  • network_region (query)

    Network region to get data for

  • fueltech (query)

    Fueltechs to get data for

  • fueltech_group (query)

    Fueltech groups to get data for

  • primary_grouping (query)

    Primary grouping to apply

  • secondary_grouping (query)

    Optional secondary grouping to apply

Responses

  • 200 Successful Response
  • 422 Validation Error
